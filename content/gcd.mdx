For finding the greatest common divisor (GCD) of two given numbers.

Brute Force Method:

<div style={{ fontSize: "1.25rem", marginBottom: "1rem" }}>
  <div>1. Identify the smaller of the two numbers.</div>
  <div>2. Iterate from the smaller number down to 1.</div>
  <div>3. In each iteration, check if the current number evenly divides both numbers (a) and (b).</div>
  <div>4. The first number that meets this condition is the GCD, which can then be returned.</div>
</div>

```java
public int findGCD_bruteForce(int a, int b) {
  int gcd = 1; // Default GCD is 1
  int smaller = Math.min(a, b);
  for (int i = smaller; i >= 1; i--) {
    if (a % i == 0 && b % i == 0) {
        gcd = i;
        break; // We found the GCD, no need to continue
    }
  }
  return gcd;
}
```

Optimized Method:

<div style={{"fontSize": "1.25rem", marginBottom: "1rem"}}>
<div>1. Divide the larger number by the smaller number and obtain the remainder.</div>
<div>2. If the remainder is 0, the smaller number is the GCD.</div>
<div>3. If the remainder is not 0, replace the larger number with the smaller number and the smaller number with the remainder.</div>
<div>4. Repeat this process until the remainder is 0.</div>
</div>
```java
public static int findGCD(int a, int b) {
  if (b == 0) {
    return a; // The base case
  }
  return findGCD(b, a % b);
}
```

another approach using euclidean algorithm

```java
public static int findGCD_euclideanIterative(int a, int b) {
  while (b != 0) {
    int temp = b;
    b = a % b;
    a = temp;
  }
  return a;
}
```
